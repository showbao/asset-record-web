# v8.4.0 使用者設定

1. 在 GitHub Pages 按「使用 Google 帳號登入」。
2. 新使用者按「建立新的資產記錄」，閱讀隱私說明後建立範本副本。
3. 開啟副本，重新整理一次讓「資產記錄」選單出現。
4. 選擇「資產記錄 → 首次建置」並完成 Google 授權。
5. 回到 GitHub Pages，貼上該副本網址並按「驗證並連結」。

已有 v8.4 正式 Sheet 的使用者可直接貼上網址。Gateway 只接受目前登入帳號可編輯、具有 `APP_ID=ASSET_RECORD`、版本可辨識、已完成首次建置且非備份的 Google Sheet。

連線只保存在目前瀏覽器的 `asset-record.connection.<Google sub>`。換瀏覽器或清除網站資料後只需重新連結，不需重建資料。
